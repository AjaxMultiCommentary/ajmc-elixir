    get_words_for_range(words, range)

    Stringifies the given range of words. Also accepts a tuple of [first, last] instead of a range.

    examples

     Examples 

    iex> get_words_for_range([%{"text" => "Ajax"}, %{"text" => "Tecmessa"}, %{"text" => "Eurysaces"}], 0..2)
"Ajax Tecmessa"

iex> get_words_for_range([%{"text" => "Ajax"}, %{"text" => "Tecmessa"}, %{"text" => "Eurysaces"}], [0, 2])
"Ajax Tecmessa"
